# TextCryption-1.2


TextCryption is a program that is used to use encrypted text files.
You can import existing text files and then use them encrypted, etc. 

TextCryption ist ein Programm, dass zum Verwenden verschlüsselter Textdateien dient.
Man kann vorhandene Textdateien importieren anschließend verschlüsselt verwenden etc.
